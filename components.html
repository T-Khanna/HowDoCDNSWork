Architectural Components in CDNs


There are many components in CDNs which make up the architecture for the CDN. These include:

  • Origin Server
  • Distribution System
  • Request Routing System
  • Account System

Origin Server

This is simply the server which originally contained the information the user is trying to access. The information from here is replicated into the CDN's surrogate cache servers.
